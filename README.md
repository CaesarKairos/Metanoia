## 📋 Sobre o Projeto

**Metanoia** é um jogo roguelike de corrida infinita (*endless runner*) desenvolvido em 2D com uma estética sombria em pixel art. O projeto foi criado de forma totalmente autoral e independente, unindo a velocidade clássica dos jogos de corrida com a imprevisibilidade e progressão de um bom roguelike.

> ⚠️ **Nota:** Este projeto está atualmente em fase inicial de **desenvolvimento** e todas as etapas de criação estão sendo documentadas no YouTube!
